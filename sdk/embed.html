Mini Plausible — Embed Demo

This page demonstrates how to integrate the Mini Plausible tracking SDK into your website.

1. Basic Script Tag

Add this snippet before </body>:

<script
  defer
  data-domain="yourdomain.com"
  data-api-endpoint="https://analytics.example.com/api/event"
  src="https://analytics.example.com/tracker.js"
></script>

2. Inline Script (No External File)

<script>
  (function() {
    var API = '/api/event';
    var SITE = 'yourdomain.com';

    function track(name, props) {
      var payload = {
        n: name,
        u: location.href,
        d: localStorage.getItem('mp_did') || crypto.randomUUID(),
        r: document.referrer,
        sw: screen.width,
        sh: screen.height,
        tp: screen.width <= 600 ? 'mobile' : 'desktop',
        timestamp: Date.now()
      };
      if (props) payload.props = props;
      navigator.sendBeacon(API, new Blob([JSON.stringify(payload)], {type:'application/json'}));
    }

    track('pageview');
    window.plausible = track;
  })();
</script>

4. Auto-Initialize

The SDK auto-detects data-* attributes from the script tag and initializes itself. No manual setup needed.

<script
  defer
  data-domain="myapp.com"
  data-api-endpoint="https://analytics.myapp.com/api/event"
  data-hash-mode="false"
  src="/tracker.js"
></script>

Features

  • Auto pageview tracking on page load
  • UTM parameter extraction
  • Device ID persistence via localStorage
  • Event batching for network efficiency
  • sendBeacon with XMLHttpRequest fallback
  • Outbound link & custom event support
  • Privacy-first: no cookies, GDPR compliant
